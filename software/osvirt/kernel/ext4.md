# Ext4

## 磁盘布局

```
       +-------------+-----------+-----------+-----------+-----------+
       | 引导块(1KB) |   块组0   |   块组1   |    ...    |   块组N   |
       +-------------+-----------+-----------+-----------+-----------+
                     |           |
                     |           |
    +----------------+           +---------------------------------------------------------+
    |                                                                                      |
    v                                                                                      v
    +--------+--------------+------------------+------------+-----------+---------+--------+
    | 超级块 | 块组描述符表 | 预留块组描述符表 | 数据块位图 | inode位图 | inode表 | 数据块 |
    +--------+--------------+------------------+------------+-----------+---------+--------+
```
